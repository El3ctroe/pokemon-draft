📖 Firebase Setup Tutorial

Firebase gives you a free real-time database so all your friends' screens sync instantly. Setup takes about 5 minutes — no coding required.

1
Go to Firebase Console
Open console.firebase.google.com and sign in with your Google account.
2
Create a New Project
Click Add Project. Give it any name like "pokemon-draft". Disable Google Analytics (not needed). Click Create Project.
3
Add a Web App
On the project overview page, click the </> web icon. Name it anything. Do NOT enable Firebase Hosting. Click Register App.
4
Copy the Config
Firebase will show you a firebaseConfig object. Copy the values inside it. Skip past the "npm install" steps and click Continue to Console.
apiKey: "AIzaSy..."
authDomain: "your-project.firebaseapp.com"
databaseURL: "https://your-project-default-rtdb.firebaseio.com"
...
5
Enable Realtime Database
In the left sidebar, click Build → Realtime Database. Click Create Database. Choose a region (any). When asked about security rules, choose Start in test mode. Click Enable.
6
Paste Config into this App
Back in this app, click Configure Firebase on the home screen and paste each value from the config object into the matching field. Click Save & Test Connection.
⚠️ Note on security: "Test mode" means anyone with your room code can join. That's fine for a friend group! The free Spark plan gives you 1GB of data and 100 simultaneous connections — way more than you'll ever need for a draft night. Test mode rules expire after 30 days; to extend them, go to Realtime Database → Rules and reset the expiry date.
